# Дизайн-система "Lumina Essence" (Atmospheric & Premium)

Этот документ описывает визуальный язык проекта. Мы отказываемся от стандартных "бутрапоподобных" решений в пользу уникальной, обволакивающей атмосферы "тихой роскоши".

## 1. Философия Визуала

- **Wabi-Sabi & Quiet Luxury:** Красота в несовершенстве натуральных материалов. Отказ от чистого белого (`#FFFFFF`) и чистого черного (`#000000`) цветов.
- **Tactile Digital:** Сайт должен ощущаться тактильно приятным, как качественная бумага или теплое стекло. Используем шум (noise textures), мягкие тени и размытия.
- **Editorial Layout:** Верстка, вдохновленная глянцевыми журналами (Kinfolk, Cereal). Смелая типографика, много воздуха, наслоение элементов.

## 2. Цветовая Палитра (Natural & Earthy)

Мы используем сложные, припыленные оттенки.

### Основа (Base)

- **Paper Warm (`#F9F8F6`)** — Основной фон. Цвет дорогой дизайнерской бумаги. Не слепит глаза, создает уют.
- **Warm Stone (`#EBE9E4`)** — Вторичный фон для секций (например, футер или карточки).
- **Soft Charcoal (`#2C2B29`)** — Основной цвет текста. Мягкий графит, не резкий черный.

### Акценты (Accents)

- **Burnt Clay (`#AA6B57`)** — Основной акцент (кнопки, ссылки). Цвет обожженной глины или сухого кленового листа.
- **Dried Sage (`#8A9A87`)** — Вторичный акцент (иконки, бейджи "Eco"). Спокойный шалфейный цвет.
- **Deep Moss (`#4A6741`)** — Глубокий зеленый для активных элементов или темных режимов.
- **Flame Gold (`#D4AF37`)** — _Использовать минимально._ Тонкие линии или обводки, имитирующие свет свечи.

### Градиенты (Atmosphere)

- _Candle Glow:_ Радиальный градиент от прозрачного к `Burnt Clay` с низкой непрозрачностью для фоновых подсветок.

## 3. Типографика (Editorial Combination)

Контраст между элегантной классикой и современным минимализмом.

### Заголовки (Headings): **Cormorant Garamond**

- _Характер:_ Изысканный, с высоким контрастом штрихов, "журнальный".
- _Использование:_ H1, H2, H3.
- _Особенность:_ Активное использование _Italic_ начертания для акцентов в заголовках (например, "Свечи, _создающие_ уют").
- _Настройки:_ `font-weight: 300/400`, `letter-spacing: -0.02em`, `line-height: 1.1`.

### Основной текст (Body): **Manrope** или **Outfit**

- _Характер:_ Геометричный, чистый, современный гротеск. Обеспечивает отличную читаемость даже в мелком кегле.
- _Использование:_ Параграфы, кнопки, интерфейсные элементы.
- _Настройки:_ `font-weight: 300/400`, `letter-spacing: 0.02em` (для воздуха), `line-height: 1.6`.

## 4. Графические Элементы и UI

### Кнопки (Buttons)

Не стандартные прямоугольники.

- **Primary:** Скругление `rounded-full` (овал) или мягкий квадрат `rounded-xl`. Фон `Soft Charcoal` или `Burnt Clay`. Текст белый. При наведении — _не_ просто затемнение, а мягкое масштабирование (`scale-105`) и легкое свечение (`shadow-glow`).
- **Secondary:** Прозрачный фон, тонкая обводка `border-[1px]`, при наведении заполняется цветом.
- **Link:** Подчеркивание появляется плавно слева направо при наведении.

### Карточки (Cards)

- Нет явных границ (`border-none`).
- Легкая, едва заметная тень (`shadow-sm`) или вообще без нее на светлом фоне.
- Картинки имеют соотношение сторон `4:5` (портретная ориентация) или `1:1`.
- При наведении на карточку фото масштабируется внутри контейнера (Zoom effect).

### Текстуры (Noise)

- На весь сайт накладывается едва заметный слой `noise.png` с `opacity: 0.03` и режимом наложения `overlay`. Это убирает "пластиковость" цифрового дизайна.

## 5. Анимации (Motion Design)

Движение должно быть медленным, плавным, "тягучим".

- **Scroll Reveal:** Элементы не просто "выпрыгивают". Они плавно проявляются (`opacity: 0 -> 1`) и немного поднимаются (`y: 20px -> 0`) с задержкой (Stagger effect).
- **Parallax:** Фоновые изображения движутся медленнее контента.
- **Images:** При загрузке изображения проявляются через эффект "размытия" (`blur-lg -> blur-0`).
- **Custom Cursor (Опционально):** Небольшой круг, который инвертирует цвет фона под собой, или "магнитится" к кнопкам.

## 6. Сетка и Композиция

- Использование **CSS Grid**.
- Асимметрия: Изображения могут быть смещены относительно центра, текст может "наезжать" на фото.
- Много негативного пространства (whitespace). Отступы `py-24` или даже `py-32` для разделения секций.

## 7. Пример UI Kit (Tailwind Config Preview)

```javascript
// tailwind.config.ts preview
theme: {
  extend: {
    colors: {
      background: '#F9F8F6',
      foreground: '#2C2B29',
      primary: '#AA6B57', // Burnt Clay
      secondary: '#EBE9E4', // Warm Stone
      accent: '#8A9A87',    // Dried Sage
    },
    fontFamily: {
      serif: ['var(--font-cormorant)', 'serif'],
      sans: ['var(--font-manrope)', 'sans-serif'],
    },
    backgroundImage: {
      'noise': "url('/noise.png')",
    }
  }
}
```
